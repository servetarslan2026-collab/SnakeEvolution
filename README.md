# 🐍 Snake Evolution

Modern roguelite Snake oyunu. Tek dosya, sıfır bağımlılık.

## 🎮 Özellikler

- **Roguelite mekanikler** — Level atla, upgrade seç, boss kes
- **8 yem türü** — Elma, altın elma, kristal, kalp, saat, mıknatıs, yıldız, coin
- **8 upgrade** — Hız, can, mıknatıs, 2x puan, 2x XP, kalkan, uzun kuyruk, ateş kuyruk
- **Düşmanlar** — Böcek, takipçi robot, hayalet
- **Boss sistemi** — Her 5 levelde boss savaşı
- **Combo sistemi** — x2, x3, x5, x10 çarpanlar
- **Neon/cyberpunk tasarım** — Gradient, glow, parçacık efektleri
- **Profesyonel font** — Orbitron (başlıklar), Rajdhani (metin)
- **78 FPS** — Akıcı performans

## 🚀 Nasıl Oynanır

1. `index.html` dosyasını tarayıcınızda açın
2. Ok tuşları veya WASD ile hareket edin
3. Yemleri yiyin, level atlayın, upgrade seçin
4. Boss'ları yenin, yüksek skor yapın

## 🛠️ Teknik

- **Dil:** Vanilla JavaScript (ES6+)
- **Görsel:** Canvas 2D API
- **Ses:** Web Audio API
- **Font:** Google Fonts (Orbitron, Rajdhani)
- **Bağımlılık:** Yok
- **Dosya:** Tek HTML dosyası
